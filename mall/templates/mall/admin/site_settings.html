📩 Messaging (Nalo — SMS + OTP)

Nalo delivers OTP codes and SMS alerts — sign-up/login codes, order alerts, and rider handoff codes — over SMS to customers, admins, officers and riders. When enabled here, Nalo is used ahead of Tiliow for SMS delivery.
  1. Create an account with NaloSolutions and top up.
  2. Copy your username and password from the Nalo dashboard into the fields below.
  3. (Optional) Set a Sender ID once approved by Nalo.
  4. Tick Enable Nalo messaging and save.

Master switch. When on, Nalo handles SMS/OTP ahead of Tiliow. When off, Tiliow (if enabled) or email/in-app notifications are used instead.

Stored securely; leave unchanged to keep the current password.

Leave as the default unless Nalo gives you a different endpoint.

Approved sender name shown to recipients (Nalo enforces an 11-character limit).

💬 Messaging (Tiliow — WhatsApp + SMS + OTP)

Tiliow delivers every phone message — login OTPs, order alerts, package-processing alerts to officers, and rider/handoff codes — over WhatsApp and SMS to customers, admins, officers and riders.
  1. Create an account at tiliow.com and top up.
  2. Copy your API key from the Tiliow dashboard into the field below.
  3. (Optional) Set a Sender ID / approved WhatsApp sender.
  4. Tick Enable Tiliow messaging and save.

Master switch. When off, the site runs on email + in-app notifications only.

From your Tiliow dashboard. Stored securely; leave unchanged to keep the current key.

Leave as the default unless Tiliow gives you a different endpoint.

Approved sender name shown to recipients, if you have one.

Digits only with country code, no plus sign. e.g. 233591784205.
